Interface(data)

new Interface(data)

Internet of Coins main API interface object. It connects to servers running Internet of Coins Hybridd using the REST API. For reference: Hybridd API
Parameters:
Name Type Description
data Object one of the three methods below must be passed:
Properties
Name Type Description
http Object a http object. (https://nodejs.org/api/http.html)
XMLHttpRequest Object a XMLHttpRequest object. (https://developer.mozilla.org/nl/docs/Web/API/XMLHttpRequest)
custom function a custom connector method which receives a hostname, query, dataCallback and errorCallback parameters and returns a string.
Source:
  • interface.js, line 88
Examples
// Node JS
var IoC = require('./ioc.nodejs.client');

var ioc = new IoC.Interface({http:require('http')});

function onSucces(){
 console.error('Done.');
}

function onError(){
  console.error('Oops, something went wrong!');
}

ioc.init(null, onSucces, onError);

sequential(data, dataCallback, errorCallback, progressCallback)

Sequentually executes functions and passes results to next step.
Parameters:
Name Type Description
data Array.<(string|Object|function())> Sequential steps to be processed. An object indicates data that is supplied to the next step. A function is a transformation of the data of the previous step and given to the next step. A string is a method that used the data from the last step and supplies to the next step.
dataCallback function Called when the method is succesful. Passes result data.
errorCallback function Called when an error occurs. Passes error.
progressCallback function Called whenever there is a progress updated. Passes a number between 0 and 1.
